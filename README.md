# **Bread Partners SDK – iOS Library**  

## **Overview**  
The **Bread Partners SDK** is a native **Swift Package Manager (SPM)-based iOS** library designed to integrate **Buy Now, Pay Later (BNPL)** functionality into brand partner applications. This SDK enables seamless interaction between native iOS applications and Bread Financial’s payment services by rendering interactive **HTML-based UI elements** and handling in-app transactions securely.  

---

## **Key Features**  
✅ **BNPL Integration** – Provides a simple way for brand partners to offer "Buy Now, Pay Later" payment options.  
✅ **Dynamic UI Rendering** – Displays **Text views** with embedded **interactive buttons**.  
✅ **Popup & WebView Support** – Opens popups containing additional content or a **secure WebView** for completing transactions.  
✅ **Secure Payment Processing** – Directs users to a **Bread Financial**  website for transactions.  
✅ **Real-Time Event Handling** – Sends **success/failure callbacks** to the brand’s native app for tracking user actions.
